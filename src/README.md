# Projeto em C++

Este é um projeto simples feito para praticar programação em C++.

O programa pede um número ao usuário e informa se ele é par ou ímpar.

## O que foi utilizado

- Entrada de dados com `std::cin`
- Saída de dados com `std::cout`
- Estrutura condicional `if/else`
- Operador matemático `%` (resto da divisão)

## Como compilar

No terminal, dentro da pasta do projeto:

g++ src/main.cpp -o programa

## Como executar

./programa

---

Projeto criado para aprendizado e prática da linguagem C++.
